Bienvenido al Ejercicio del Segundo Parcial

Este proyecto fue desarrollado por Jesús Felipe Garibay Urías como parte de una práctica con Angular. La aplicación consta de cinco páginas que permiten gestionar distintos elementos.

¿Qué encontrarás en esta aplicación?

  • Inicio: Página principal con información sobre el proyecto.
  • Productos: Gestión de productos con opciones para agregar, modificar y eliminar.
  • Libros: Catálogo de libros con funcionalidad CRUD.
  • Ejercicio: Sección de ejercicios prácticos.
  • Acerca de: Información adicional sobre el proyecto.

Características del Proyecto

Esta aplicación fue creada utilizando:

  • Angular CLI versión 19.0.7
  • Bootstrap para el diseño responsivo
  • Firebase para el almacenamiento de datos
  • Componentes modulares para facilitar la escalabilidad

¿Cómo utilizar la aplicación?

Para navegar entre las secciones, usa el menú en la parte superior. Cada sección cuenta con opciones interactivas para gestionar la información.
